"Como clonar WhatsApp com QR Code?

20-35 Minutes

Vá até o site – A primeira coisa a se fazer é entrar no site: https://web.whatsapp.com/. Assim que tiver o acesso, você verá a tela a seguir:

Escaneie o QR code – Após isso, você irá visualizar um mini tutorial onde estará escrito o que fazer no momento. Basta seguir os passos:

  • Abra o Whatsapp no seu celular

  • Toque em Mais opções ou Ajustes e selecione WhatsApp Web

  • Aponte seu celular para essa tela para capturar o código

Pronto, o seu, ou o Whatsapp de quem você planeja clonar, já está na tela do computador e, mesmo que fechar a aba, na próxima vez que você reabri-la, ele estará lá.

De todos os métodos, esse é o mais simples, dependendo apenas de que o Whatsapp que você busca clonar seja conectado via web no seu computador, para que assim fique salvo no navegador.
